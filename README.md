# Mark-s-Dungeon
Definetly not inspired by "Dungeon Crawler Carl" book series

This is a 2.5D dungeon game, using Unity 6.3.

*Current Status:* *WIP*
Initializing the project and starting with core logic. Create the project structure and the base for enemies and player scripts.
Added ScriptableObjects for:
* Enemy types
* Player type (class)
* Skills (will be used by player, enemies and companion)

Created a base input system.
